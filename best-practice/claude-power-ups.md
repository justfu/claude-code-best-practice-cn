# Power-ups 最佳实践

![Last Updated](https://img.shields.io/badge/Last_Updated-Apr%2002%2C%202026-white?style=flat&labelColor=555)

```bash
claude

```

## Power-ups（10 个）

<p align="center">
  <img src="assets/claude-power-ups/powerup-menu.png" alt="显示 10 个课程的 Power-ups 菜单" width="700">
</p>

| # | Power-up | 主题 |
|---|----------|--------|
| 1 | 与代码库对话 | `@` 文件引用、行引用 |
| 2 | 用模式操控 | `shift+tab`、plan、auto |
| 3 | 撤销任何操作 | `/rewind`、`Esc-Esc` |
| 4 | 后台运行 | tasks、`/tasks` |
| 5 | 教 Claude 你的规则 | `CLAUDE.md`、`/memory` |
| 6 | 用工具扩展 | MCP、`/mcp` |
| 7 | 自动化你的工作流 | skills、hooks |
| 8 | 分身术 | subagents、`/agents` |
| 9 | 从任何地方编码 | `/remote-control`、`/teleport` |
| 10 | 调节模型 | `/model`、`/effort` |
